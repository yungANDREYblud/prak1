Что такое CMS

CMS - " система управления контентом" (движок) - написанная РНР- программистами основа для сайтов.с помощью которой вы сможете управлять сайтом (добавлять контент, менять пункты меню и т,п) не зная HTML и CSS. Однако, для того чтобы сделать сайт с помощью CMS потребуются услуги программиста, и дизайнера и верстальщика.

Какие бывают cms

Бывают различные системы управления контентом: для интернет-магазин, для блого, для форумов и т.д.

Примеры cms

Примеры популярных CMS:Joomla, WordPress (для блогов), PhpBB (для форумов)

CMS-ки бывают платные и бесплатные.
