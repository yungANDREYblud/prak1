PHP и JavaScript

Языки прграммирования PHP и JavaScript позволяют сделать сайт динамичным, то есть реагирующим на действия пользователя. Например, можно сделать красивсую выпадающию менюшку или слайдер

Виды скриптов

Для этого пишутся скрипы (англ. script - "сценарий") - программы, позволяющие реагировать на действия пользователя.
  • те,которые выполняются на сервере, а результат их выполнения приходит в браузер к пользователю уже в готовом виде. Это скрипы написанные на языке РНР. На нем пишутся CMS-ки - система управление контентом.
  • те, которые выполняются прямов в браузер пользователя. Это скрипты на языке JavaScript. Они чаще всего используются для того, чтобы сделать страницу более удобной и красивой.
